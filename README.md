## 課題の提出方法
* 毎回の課題で、必ず提出しないといけないものは、次の二点。
  1. **共有URL**: 授業で指定したPythonコード（Google Colabノートブックの共有リンク）
  2. **作業経過レポート**: そのPythonコードを書いている途中で生じた、以下の3種類の出来事のを、3件。（PDF文書で。）
- [x] 「私が書いたコードを私自身はこう修正した」
  - 文法エラーを修正した、あるいは、より上手いコードを書けるようになったという、自分自身の気づきの報告
- [x] 「私が書いたコードを生成AIはこう修正した」
  - AIに助けてもらって、文法エラーを修正した、あるいは、効率の悪いコードを効率化した、等の報告
- [x] 「生成AIが書いたコードを私はこう修正した」
  - 課題の意図に合わないコードをAIが生成したので自分で修正した、等の報告
* 「生成AIが書いたコードを生成AIはこう修正した」という種類の報告は、**カウントしません**。3件に含ませることはできません。
